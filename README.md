# DigiTox

Control your time. Control your life. 

DigiTox is a smart digital detox web application that helps you overcome screen addiction through behavioral psychology and intelligent interventions. It provides a comprehensive set of tools to track, manage, and reduce your screen time while building healthier digital habits.

## Features

- **Dashboard**: Get a quick overview of your daily digital habits and screen time.
- **Focus Mode**: Minimize distractions and stay focused on your tasks.
- **Insights**: Analyze your behavior patterns through detailed charts and metrics.
- **Habits**: Build and maintain healthy digital boundaries.
- **Emergency Lock**: Instantly block distractions when you need deep focus.

## Tech Stack

- **Frontend**: Vanilla HTML, CSS, and JavaScript.
- **Bundler**: [Vite](https://vitejs.dev/) for fast and modern development.
- **Data Visualization**: [Chart.js](https://www.chartjs.org/) for beautiful and interactive insights.

## Getting Started

1. Clone this repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
